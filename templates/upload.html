About TextBlob Analysis

TextBlob is a simple library for processing textual data. It provides a consistent API for diving into common natural language processing (NLP) tasks such as part-of-speech tagging, noun phrase extraction, sentiment analysis, classification, translation, and more. The sentiment analysis returns a polarity score indicating how positive or negative the text is and a subjectivity score indicating how subjective or objective the text is.
